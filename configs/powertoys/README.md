# PowerToys Configuration

## How to Import

1. Open **PowerToys → General → Backup & Restore**
2. Click **Restore** and select the `.ptb` file from this folder
   *(Generate your own first via Backup, then share across machines)*

## FancyZones Layout

**3-column wide monitor (recommended for 27"+ / ultrawide):**
```
┌──────────┬─────────────────┬──────────┐
│   30%    │      40%        │   30%    │
│ Terminal │   Main Editor   │ Browser  │
└──────────┴─────────────────┴──────────┘
```
**Hotkey:** `Win+Shift+Z` to open layout picker
**Snap:** Hold `Shift` while dragging any window

## PowerToys Run
- **Hotkey:** `Alt+Space`
- Enabled plugins: Everything, Programs, VS Code Workspaces, Calculator, Windows Settings, Shell
